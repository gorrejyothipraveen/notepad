# google note pad

- **data base** :
  - users information
  - all the notes

- users information :
  - id
  - username

- notes information :
  - id
  - note name
  - note content
  - user_id (from the users table)

- **frontend part**
  - login page
  - profile page
  - profile page it self contain the logout button

  - what login page contain :
    - username
    - password

  - what profile page consists of :
    - all the notes as a side bar
    - middle area to write something
    - logout button

  - side bar :
    - all the names of the notes created by the user
      - when he clicks on that it will navigate to that particular notes and
        allow user to edit it
    - middle area
      - which large text box
      - at the end there is a input (give name of the note and click on the
        submit button)
    - logout button
      - when user clicks on that it will takes to login page again
